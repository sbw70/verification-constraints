# CSV Evidence

This directory stores generated comparison output from the phase 1 separated-services benchmark.

CSV files in this directory are generated evidence artifacts, not source code.

Expected evidence may include:

- fixed-run comparison output
- continuous comparison samples
- latency measurements
- denial behavior measurements
- downstream activation counts
- provider-first versus conventional request-path observations

Generated CSV files should preserve enough context to support reproducible review of the benchmark behavior.
